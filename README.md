# youtube-inbox

A cleaner YouTube experience for focused watching.

The script does 2 things: 
- Hides distractions (`declutter.js`) 
- Hides watched videos (`hide-watched-videos.js`). When YouTube watch history is enabled, fully watched videos are marked with a red progress bar. But if you watched a video on another account, cleared your history, or had history disabled â€” there's no way to mark it as fully watched quickly. To work around this, the script also hides liked/disliked videos, letting you use ratings as a permanent "watched" marker.
